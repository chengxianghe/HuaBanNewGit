# HuaBanNewGit 
===
- 用 swift 实现的花瓣app的效果, 也是练习swift, 仅供学习交流使用. 
- 项目使用 ARC 和 CocoaPods 
- iOS 8.0
- swift 2.2

GitHub：[chengxianghe](https://github.com/chengxianghe) 

## Version 1.1:
- 加入了三方分享 微信,QQ,微博
- 加入图片保存到相册
- bugTags 设置成摇晃启动,默认隐藏
- 修复部分bug

## Version 1.0:
- 完成项目基本框架
- 集成了bugTags 可以直接反馈bug;
